Scheme Chat

Internet Relay Chat (IRC)

IRC is most easily visited via webchat. For regular use, it's recommended to install an IRC client on your computer. For persistent sessions so you don't get logged out of IRC when shutting down your computer, a shell account or a bouncer can be installed on a Linux or Unix server. Universities commonly provide a shell account suitable for IRC to their students and faculty. IRCCloud is a commercial bouncer with a webchat.

Scheme is a prominent member of the Lisp language family. Therefore channels dealing with the family as a whole are included on this page. Scheme and Lisp related IRC channels are on the Freenode network except where otherwise noted.

Scheme channels

  • #scheme — the main channel for the Scheme language (logs)
  • #scheme-live — fast-paced hacking on Scheme integration

Scheme implementation channels

  • #chez — Chez Scheme
  • #chicken — CHICKEN
  • #gambit — Gambit Scheme
  • #gerbil-scheme — Gerbil Scheme
  • #guile — GNU Guile
  • #ironscheme — IronScheme
  • #loko — Loko Scheme
  • #racket — Racket

Scheme tool channels

  • #akku — the Akku package manager for Scheme libraries
  • #guix — the GNU Guix package manager and Linux distro
  • #scsh — the Scsh Scheme shell

Lisp family channels

  • ##lisp — the main channel for the Lisp family as a whole (note two # signs)
  • #lispcafe — casual, beginner-friendly channel for all Lisp dialects
  • #lispgames — game programming in any Lisp dialect
  • #lisp-lab — tutoring for any Lisp dialect
